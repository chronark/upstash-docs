Either a single object or an array of objects is returned, depending on
whether you have sent the message to a url, an api, or topic

### Single Object Response

This is returned when the destination is a url or an api.

<ResponseField name="messageId" type="string" required>
  The unique id of this message.
</ResponseField>

<ResponseField name="deduplicated" type="boolean">
  Whether this message is a duplicate and was not sent to the destination.
</ResponseField>

### Array of Objects Response

This is returned when the destination is a topic, and array contains
one element for each url in the topic.

<ResponseField name="array" type="object[]">
  <Expandable defaultOpen>

    <ResponseField name="messageId" type="string" required>
      The unique id of this message.
    </ResponseField>

    <ResponseField name="url" type="string" required>
      The url where the message was sent to.
    </ResponseField>

    <ResponseField name="deduplicated" type="boolean">
      Whether this message is a duplicate and was not sent to the destination.
    </ResponseField>

  </Expandable>
</ResponseField>
