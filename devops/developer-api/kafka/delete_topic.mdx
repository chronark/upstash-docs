---
title: "Delete Kafka Topic"
description: "This endpoint creates a new kafka topic in a cluster."
api: "DELETE https://api.upstash.com/v2/kafka/topic"
authMethod: "basic"
---

<Note>
Request parameters should be encoded as JSON.
</Note>

## URL Parameters
<ParamField body="ID" type="string" required>
The ID of the Kafka Topic to be deleted
</ParamField>




<RequestExample>


```shell curl
curl -X DELETE \
  https://api.upstash.com/v2/kafka/topic/:id \
  -u 'EMAIL:API_KEY'
```

```python Python
import requests

response = requests.delete('https://api.upstash.com/v2/kafka/topic/:id' auth=('EMAIL', 'API_KEY'))
response.content
```

```go Go
client := &http.Client{}
req, err := http.NewRequest("DELETE", "https://api.upstash.com/v2/kafka/topic/:id", nil)
if err != nil {
    log.Fatal(err)
}
req.SetBasicAuth("email", "api_key")
resp, err := client.Do(req)
if err != nil {
    log.Fatal(err)
}
bodyText, err := ioutil.ReadAll(resp.Body)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("%s\n", bodyText);
```


</RequestExample>
