---
title: "Get Database Stats"
description: "This endpoint gets detailed stats of a database."
api: "GET https://api.upstash.com/v2/redis/stats/:id"
authMethod: "basic"
---
<Note>
Request parameters should be encoded as JSON.
</Note>

## URL Parameters
<ParamField body="ID" required>
The ID of the database
</ParamField>

<RequestExample>


```shell curl
curl -X GET \
  https://api.upstash.com/v2/redis/stats/:id \
  -u 'EMAIL:API_KEY'
```

```python Python
import requests

response = requests.get('https://api.upstash.com/v2/redis/stats/:id', auth=('EMAIL', 'API_KEY'))
response.content
```

```go Go
client := &http.Client{}
req, err := http.NewRequest("GET", "https://api.upstash.com/v2/redis/stats/:id", nil)
if err != nil {
log.Fatal(err)
}
req.SetBasicAuth("email", "api_key")
resp, err := client.Do(req)
if err != nil {
log.Fatal(err)
}
bodyText, err := ioutil.ReadAll(resp.Body)
if err != nil {
log.Fatal(err)
}
fmt.Printf("%s\n", bodyText);
```
