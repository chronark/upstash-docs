Upstash offers developers serverless databases and messaging platforms to build powerful applications without having to worry about the operational complexity of running databases at scale.

## Get Started

<CardGroup cols={2}>
  <Card title="Serverless">
    Upstash is serverless. You don't need to provision any infrastructure. Just
    create a database and start using it.
  </Card>
  <Card title="Scaling to Zero">
    All Upstash products scale to zero. You don't pay for idle machines or
    unused resources. You pay only for what you use.
  </Card>
</CardGroup>
