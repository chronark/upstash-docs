---
title: LREM
description: Remove the first `count` occurences of an element from a list.
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the list.
</ParamField>

<ParamField body="count" type="number" required>
 How many elements to remove.
</ParamField>


<ParamField body="element" type="TValue" required>
 The element to remove
</ParamField>

## Response

<ResponseField type="integer" required>
  The number of elements removed.
</ResponseField>

<RequestExample>
```ts Example 
await redis.lpush("key", "a", "a", "b", "b", "c"); 
const removed = await redis.lrem("key", 4, "b"); 
console.log(removed) // 2
 ```
</RequestExample>
