---
title: SADD
description: Adds one or more members to a set.
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the set.
</ParamField>

<ParamField body="members" type="...TValue[]" required>
  One or more members to push at the end of the list.
</ParamField>


## Response

<ResponseField type="number" required>
  The length of the set after the push operation.
</ResponseField>

<RequestExample>
```ts Example 
await redis.sadd("key", "a", "b", "c"); 
```

</RequestExample>
