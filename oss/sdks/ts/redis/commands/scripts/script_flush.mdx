---
title: SCRIPT FLUSH
description: Removes all scripts from the script cache.
---

## Arguments

<ParamField body="hahes" type="string[]" required>
  The sha1 of the scripts to check.
</ParamField>

## Response

<ResponseField type="(0 | 1)[]" required>
  For every script that exists, an element of the array will be 1, otherwise 0.
</ResponseField>

<RequestExample>
```ts Example
const result = await redis.scriptExists([sha]);
console.log(result) // [0] or [1] depending on if the script exists or not.

```
</RequestExample>
