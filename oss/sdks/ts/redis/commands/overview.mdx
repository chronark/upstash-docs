## String


<CardGroup cols={3}>

<Card title="APPEND" href="/oss/sdks/ts/redis/commands/string/append">
Append a value to a string stored at key.
</Card>
<Card title="DECR" href="/oss/sdks/ts/redis/commands/string/decr">
Decrement the integer value of a key by one.
</Card>
<Card title="DECRBY" href="/oss/sdks/ts/redis/commands/string/decrby">
Decrement the integer value of a key by the given number.
</Card>
<Card title="GET" href="/oss/sdks/ts/redis/commands/string/get">
Get the value of a key.
</Card>
<Card title="GETDEL" href="/oss/sdks/ts/redis/commands/string/getdel">
Get the value of a key and delete the key.
</Card>
<Card title="GETRANGE" href="/oss/sdks/ts/redis/commands/string/getrange">
Get a substring of the string stored at a key.
</Card>
<Card title="GETSET" href="/oss/sdks/ts/redis/commands/string/getset">
Set the string value of a key and return its old value.
</Card>
<Card title="INCR" href="/oss/sdks/ts/redis/commands/string/incr">
Increment the integer value of a key by one.
</Card>
<Card title="INCRBY" href="/oss/sdks/ts/redis/commands/string/incrby">
Increment the integer value of a key by the given amount.
</Card>
<Card title="INCRBYFLOAT" href="/oss/sdks/ts/redis/commands/string/incrbyfloat">
Increment the float value of a key by the given amount.
</Card>
<Card title="MGET" href="/oss/sdks/ts/redis/commands/string/mget">
Get the values of all the given keys.
</Card>
<Card title="MSET" href="/oss/sdks/ts/redis/commands/string/mset">
Set multiple keys to multiple values.
</Card>
<Card title="MSETNX" href="/oss/sdks/ts/redis/commands/string/msetnx">
Set multiple keys to multiple values, only if none of the keys exist.
</Card>
<Card title="PSETEX" href="/oss/sdks/ts/redis/commands/string/psetex">
Set the value and expiration in milliseconds of a key.
</Card>
<Card title="SET" href="/oss/sdks/ts/redis/commands/string/set">
Set the string value of a key.
</Card>
<Card title="SETEX" href="/oss/sdks/ts/redis/commands/string/setex">
Set the value and expiration of a key.
</Card>
<Card title="SETNX" href="/oss/sdks/ts/redis/commands/string/setnx">
Set the value of a key, only if the key does not exist.
</Card>
<Card title="SETRANGE" href="/oss/sdks/ts/redis/commands/string/setrange">
Overwrite part of a string at key starting at the specified offset.
</Card>
<Card title="STRLEN" href="/oss/sdks/ts/redis/commands/string/strlen">
Get the length of the value stored in a key.
</Card>
<Card title="SUBSTR" href="/oss/sdks/ts/redis/commands/string/substr">
Get a substring of the string stored at a key.
</Card>

</CardGroup>
