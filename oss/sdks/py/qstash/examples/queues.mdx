#### Create a queue with parallelism

```python
from upstash_qstash import QStash

qstash = QStash("<QSTASH-TOKEN>")

queue_name = "upstash-queue"
qstash.queue.upsert(queue_name, parallelism=2)

print(qstash.queue.get(queue_name))
```

#### Delete a queue

```python
from upstash_qstash import QStash

qstash = QStash("<QSTASH-TOKEN>")

queue_name = "upstash-queue"
qstash.queue.delete(queue_name)
```

#### Pause/Resume a queue

```python
from upstash_qstash import QStash

qstash = QStash("<QSTASH-TOKEN>")

queue_name = "upstash-queue"
qstash.queue.upsert(queue_name, parallelism=1)

qstash.queue.pause(queue_name)

queue = qstash.queue.get(queue_name)
print(queue.paused) # prints True

qstash.queue.resume(queue_name)
```
