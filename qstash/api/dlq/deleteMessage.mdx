---
title: "Delete a message from the DLQ"
description: "Manually remove a message"
api: "DELETE /v2/dlq/messages/:dlqId"
authMethod: "bearer"
---

List all messages currently inside the DLQ

## Request

<ParamField path="dlqId" type="string">
  The dlq id of the message you want to remove. You will see this id when listing all messages in the dlq with the [/v2/dlq/messages](/qstash/api/dlq/listMessages) endpoint.
</ParamField>

## Response

The endpoint doesn't return anything, a status code of 200 means the message is removed from the DLQ.

<RequestExample>

```sh
curl https://qstash.upstash.io/v2/dlq/messages/my-dlq-id \
  -H "Authorization: Bearer <token>"
```

</RequestExample>
