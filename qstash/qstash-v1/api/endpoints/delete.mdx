---
title: "Delete Endpoint"
description: "This endpoint deletes an existing endpoint from QStash. "
api: "DELETE https://qstash.upstash.io/v1/endpoints/:endpointId"
authMethod: "bearer"
---

<Note>
  This will only apply to newly created messages. All messages published before
  this endpoint was updated will still use the old url.
</Note>

## Request

<ParamField path="endpointID" type="string" required>
The id of the endpoint

</ParamField>

<RequestExample>
```shell curl
curl -X DELETE https://qstash.upstash.io/v1/endpoints/:endpointId \
  -H "Authorization: Bearer <token>"
```
</RequestExample>

<ResponseExample>

```json 200
{ "status": "ok" }
```

</ResponseExample>
